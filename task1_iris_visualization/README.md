# Task 1: Exploring and Visualizing the Iris Dataset

## Objective
The objective of this project is to explore, inspect, and visualize the Iris dataset using Python data science libraries such as pandas, seaborn, and matplotlib.

---

## Tools and Libraries Used
- Python
- Pandas
- Seaborn
- Matplotlib
- Jupyter Notebook

---

## Dataset Information
The Iris dataset contains flower measurements from three species of iris flowers:

- Setosa
- Versicolor
- Virginica

The dataset includes the following features:
- Sepal Length
- Sepal Width
- Petal Length
- Petal Width

---

## Tasks Performed
### 1. Dataset Loading
- Loaded the dataset using seaborn.

### 2. Dataset Inspection
- Checked dataset shape
- Displayed column names
- Reviewed dataset information
- Generated statistical summary

### 3. Data Visualization
Created multiple visualizations including:
- Scatter Plot
- Histograms
- Box Plot
- Pair Plot

### 4. Insights
- No missing values were found.
- Petal features strongly help distinguish flower species.
- Setosa species is clearly separable from other species.
- Most distributions are balanced with very few outliers.
